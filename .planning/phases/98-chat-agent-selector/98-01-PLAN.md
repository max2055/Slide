## Architecture Context

Chat 页面渲染分两层：
1. **app-render.ts line 939**: `renderChatSessionSelect(state)` — 渲染在 content-header 区域
   - 实现在 `app-render.helpers.ts:176` — 包含 session select + model select + thinking select
2. **app-render.ts line 1608**: `renderChat(props)` — 主聊天组件
   - 实现在 `views/chat.ts`
   - 已通过 ChatProps 暴露 agentsList / currentAgentId / onAgentChange

当前 `app-render.helpers.ts` 的 `renderChatSessionSelect` 只有 Session 选择器（和 Model/Thinking），**没有** Agent 选择器。
移植方案：直接在已有的 `renderChatSessionSelect` 函数头部调用 `renderChatAgentSelect`，把 Agent 下拉框渲染在 Session 下拉框前面。
