# Plan 96-04: Fix sslOptions / fetchAs Module-Level Mutation Blocker

**Gap from UAT:** Test 1 — `Cannot add property sslOptions, object is not extensible`

**Root cause:** oracledb Thin mode (v6.x+) freezes the module singleton. Direct property assignment (`oracledb.sslOptions = ...`, `oracledb.fetchAsString = ...`) throws TypeError.

**Fix strategy:** Remove all module-level property assignments. Pass `fetchAsString` and `fetchAsBuffer` via `createPool()` options. Drop `sslOptions` entirely — Thin mode TCPS handles TLS negotiation automatically without the `rejectUnauthorized` flag.
