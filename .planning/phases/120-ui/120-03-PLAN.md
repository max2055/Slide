    - Empty state: renders `<app-empty-state>` as a fallback row colspan
    - CSS: standard `<table>` element with:
      - `width: 100%; border-collapse: collapse;`
      - `th`: var(--text-sm), font-weight 600, text-align left, padding var(--space-md), border-bottom 2px solid var(--border), color var(--muted-strong)

      - `.striped tbody tr:nth-child(even)`: background var(--bg-muted)
      - `tr:hover`: background var(--bg-hover)
      - `.dense th, .dense td`: padding var(--space-sm)

    Do NOT implement virtual scrolling or complex sorting logic — just the skeleton UI that standard views use. The sort event is emitted for the parent to handle.
