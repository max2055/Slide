    Create `frontend/src/app/ui/components/app-dialog.ts` — unified dialog/modal component per D-11.

    Implementation requirements:
    - Light DOM: `createRenderRoot() { return this; }`
    - Tag: `<app-dialog>`
    - Properties:
      - `open: boolean` — controls visibility
      - `size: 'sm' | 'md' | 'lg' | 'xl'` — width variants: sm=400px, md=520px, lg=640px, xl=720px
      - `title: string` — dialog title text
      - `closable: boolean = true` — show close button
    - Slots: default (body content), `footer` (action buttons)
    - Animation: scale-in + fade-in, 200ms, var(--ease-out)
    - Overlay: semi-transparent background (`rgba(0,0,0,0.3)`), z-index: var(--z-modal, 1000)
    - ESC key closes dialog (if closable)
    - Click on overlay closes dialog (if closable)
    - Focus trap: on open, focus first focusable element; on ESC, return focus to trigger element
    - Inner dialog box: max-width from size prop, background var(--card), border-radius var(--radius-lg), box-shadow var(--shadow-xl)
    - Title: var(--text-lg), font-weight 600
    - Close button: top-right, 32x32px, uses renderIcon() for X icon (or inline SVG path for times/close)
    - Events: dispatches 'app-dialog-close' CustomEvent on close
    - The component renders nothing when `open` is false (return `nothing` from lit-html)

    Match the component styles to the existing patterns: use --space-lg for padding, --border for overlay border (if any), --radius-lg for corners.

    CSS structure: render the overlay + dialog + close button in a single template.
    ```
    <div class="dialog-overlay" @click=${this._onOverlayClick} ?hidden=${!this.open}>
      <div class="dialog" role="dialog" aria-modal="true" aria-label=${this.title}>
        <div class="dialog-header">
          <span class="dialog-title">${this.title}</span>
          ${this.closable ? html`<button class="dialog-close" @click=${this._close}>×</button>` : nothing}
        </div>
        <div class="dialog-body"><slot></slot></div>
        <div class="dialog-footer"><slot name="footer"></slot></div>
      </div>
    </div>
    ```
